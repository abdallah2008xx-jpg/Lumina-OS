# Lumina-OS Blocker Records

Store blocker reviews here after syncing the latest VM-cycle results.

## Recommended Flow
1. Finish the VM cycle with `scripts/finish-vm-test-cycle.ps1`
2. Review `status/blockers/CURRENT-BLOCKERS.md`
3. Open the dated blocker review under `status/blockers/YYYY-MM-DD/`
4. Fix blockers or close the attention items
5. Rerun `scripts/sync-test-blockers.ps1` if you want a refreshed blocker state after edits

- VM report blockers
