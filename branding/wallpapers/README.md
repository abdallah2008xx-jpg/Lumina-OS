# AhmadOS Wallpaper Pack 01

This folder contains the first AhmadOS wallpaper direction pack.

## Files
- `ahmados-horizon.svg` - bright release wallpaper with layered mineral forms
- `ahmados-lagoon.svg` - blue-forward wallpaper for desktop and welcome surfaces
- `ahmados-nocturne.svg` - darker wallpaper suited to login and evening mode

## Direction
- calm gradients
- strong horizon and landscape cues
- muted copper highlights
- enough contrast for desktop icons and login overlays

## Notes
- SVG is used for the first pass so the direction stays editable
- export PNG variants later for release packaging and SDDM usage
