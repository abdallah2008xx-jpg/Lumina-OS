# Office and Productivity Launch Recipe

## Daily Start

- Start with Lumina-native mail, browser, notes, and file management first.
- Escalate into the Windows workspace only when the document or collaboration workflow genuinely needs it.
- Keep the work session narrow: one office app family at a time.

## Daily Rhythm

- Treat the Lumina side as the source of truth for files, meetings, and notes.
- Use the Windows side only for the office tool that still blocks the Linux-first flow.
- Validate save, export, and re-open behavior before trusting the workflow for important documents.

## Daily Close

- Export final documents back into the Lumina workspace.
- Keep meeting notes and work logs on the Linux side so the workflow stays easy to recover.
