# Lumina-OS Shareable Updates

Store generated public-facing progress updates here.

## Intended Contents
- dated snapshots of the current project state
- a generated shareable update rooted in readiness, validation, and release-candidate status

## Recommended Flow
1. Refresh project status files first
2. Run `scripts/sync-shareable-update.ps1`
3. Review `status/SHAREABLE-UPDATE.md`
4. Share the current update externally if the wording looks right
