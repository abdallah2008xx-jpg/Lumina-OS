# AhmadOS Prototypes

This folder contains local HTML prototypes for AhmadOS interface direction.

## Pages
- `index.html` - prototype gallery
- `welcome/` - first-run welcome app
- `update-center/` - update center shell
- `settings/` - settings shell
- `login/` - SDDM login direction

## Notes
- These are visual and structural prototypes, not production code
- They are designed to reflect the brand system and UX documents in `docs/` and `branding/`
- Open `index.html` in a browser to navigate the set
