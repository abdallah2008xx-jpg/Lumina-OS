# Adobe Creator VM Template Notes

## Core Shape

- Keep one stable Windows VM for the whole Adobe stack.
- Favor SSD-backed storage, predictable cache placement, and clean shared paths.
- Treat this as a creator workstation template, not a throwaway VM.

## Base Priorities

- Install Creative Cloud first, then Photoshop and Illustrator, then the heavier video tools.
- Keep fonts, plugins, and export folders organized from day one.
- Validate one clean project import and one clean export back to Lumina early.
