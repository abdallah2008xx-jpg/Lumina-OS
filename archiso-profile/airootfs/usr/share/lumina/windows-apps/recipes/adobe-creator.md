# Adobe Creator Launch Recipe

## Daily Start

- Start from Lumina first, then verify the creator workspace folders are ready before opening Windows-side tools.
- Launch the Windows workspace and bring up Creative Cloud before the heavier Adobe apps.
- Open only the app stack you need for the session so fonts, caches, and plugins stay predictable.

## Daily Rhythm

- Keep ingest, references, communication, and exports visible on the Lumina side.
- Keep the heavier creator apps inside one Windows workspace so the workflow feels like one studio instead of many disconnected islands.
- Validate one clean export back into Lumina early in the session.

## Daily Close

- Close Adobe apps in reverse order of use, then wait for caches, exports, and sync work to finish.
- Keep the final deliverables on the Lumina side and record plugin or media issues before ending the day.
