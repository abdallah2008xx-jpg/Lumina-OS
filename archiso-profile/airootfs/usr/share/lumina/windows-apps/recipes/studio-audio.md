# Studio and Audio Launch Recipe

## Daily Start

- Prepare sample paths, project notes, and storage before opening the Windows-side audio tools.
- Launch the Windows workspace only after the shared paths are ready and visible.
- Open the DAW or plugin host in a focused session instead of mixing unrelated tasks.

## Daily Rhythm

- Keep the Lumina side responsible for notes, imports, exports, and general system flow.
- Keep plugin-heavy audio work inside one stable Windows workspace.
- Validate one sample path, one plugin chain, and one export path early in the session.

## Daily Close

- Export renders or stems back into the Lumina side before closing the session.
- Record stable plugin combinations and trouble spots before shutdown.
